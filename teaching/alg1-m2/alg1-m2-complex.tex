\section{Complex Numbers}

Given a complex number $z = x + yi$, the \emph{magnitude} of $z$, denoted $\lvert z\rvert$, is given by $\sqrt{x^2 + y^2}$. For example,
\begin{equation*}
\lvert -1 + 3i\rvert = \sqrt{(-1)^2 + 3^2} = \sqrt{10}.
\end{equation*}
